school-walker
===
### 项目说明
利用重要信息维护的XSS漏洞，获得用户cookie后，使用本项目存储数据并进行功能的拓展。

### 已实现功能
 - 高仿请假页面(和真正请假页面完全一样)
 - 保存用户数据以便一键签到
### 待实现
 - 一键签到(之前做过简易版本的)
 - 自定义请假时间(目前请假时间是当天8时到晚上20时，以后会考虑让用户自定义)
